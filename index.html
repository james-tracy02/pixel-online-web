Controls:

Draw - Left Click
Pick Color - Right Click
Pick Target Color - Ctrl + Right Click
Change Pen Size - QE
Move Canvas - WASD
Zoom - Mouse Wheel


Modes:

Normal - Overwrite existing pixels regardless of their color.

Coloring - Overwrite all pixels except those which match target color. (Useful for filling in line-art.)

Replace - Overwrite only pixels which match target color. (Useful for mass-replacing a color.)
